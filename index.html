Nevernote (personal project)

http://never-note.herokuapp.com/

https://github.com/geehaws/Nevernote

This is an Evernote clone I built in Javascript on the Backbone.js framework. It is served by RESTful Rails API and utilizes JQuery's UI plugin to create an interactive user experience. It is a 100% asynchronous single-page app that allows for WYSIWYG text editing, version tracking, and image uploading. Stores uploaded files with AWS S3.
