Here are some of the bigger projects I am working on:

Nevernote

http://never-note.herokuapp.com/

https://github.com/geehaws/Nevernote

This is an Evernote clone I built in Javascript on the Backbone.js framework. It is served by RESTful Rails API and utilizes JQuery's UI plugin to create an interactive user experience. It is a 100% asynchronous single-page app that allows for WYSIWYG text editing, version tracking, and image uploading.

Blurred

https://github.com/geehaws/Blurred

This is a Newsblur clone, also built in Javascript on Backbone and served by a Rails API. It is an asynchronous single page RSS reader that allows the user to view external web pages within a frame in the app; no need to leave the page! It allows for seamless transitions between in-app views and external articles.
